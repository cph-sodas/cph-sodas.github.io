# Documentation for the SODAS server

## Installation

```bash
uv venv
source .venv/bin/activate  # linux & mac
uv sync
```

## Commands

- `mkdocs serve` - Start the live-reloading docs server.
- `make deploy` - Deploy the documentation to GitHub pages.
